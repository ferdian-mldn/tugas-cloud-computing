# Implementasi PaaS Cloud Computing

Repository ini berisi source code untuk memenuhi Tugas Ujian Akhir Semester (UAS) mata kuliah Cloud Computing. Proyek ini mendemonstrasikan implementasi layanan Platform as a Service (PaaS) menggunakan infrastruktur Vercel dengan integrasi CI/CD (Continuous Integration / Continuous Deployment) langsung dari GitHub.

## Deskripsi Proyek
Proyek ini merupakan sebuah antarmuka web statis bergaya modern (dark-mode) yang dirancang menggunakan utility-first CSS framework (Tailwind). 

Tujuan utama dari proyek ini adalah untuk membuktikan pemahaman praktis terkait arsitektur cloud computing. Dengan menggunakan Vercel, developer tidak perlu lagi mengonfigurasi peladen (server) fisik secara manual. Vercel bertindak sebagai platform awan publik yang mengotomatisasi proses build, mengatur sertifikat keamanan (SSL) secara instan, dan melakukan distribusi aset statis secara global melalui jaringan Edge mereka.
